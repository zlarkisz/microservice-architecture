# Частина 1: Доменна модель

## Завдання

Побудувати доменну модель для E-Commerce порталу.

---

## Крок 1: Аналіз вимог

### Вхідні дані

З опису системи виділено наступні компоненти:

| Розділ           | Опис з завдання                                          |
| ---------------- | -------------------------------------------------------- |
| Фронтенд         | Лист товарів, Кошик, Блог та новини, Відгуки та рейтинги |
| Бекенд           | Товари, Категорії, Постачальники, Бренди, Локалізація    |
| Фінанси          | Склад, Бухгалтерія, Податки, Знижки                      |
| Інтеграції       | Постачальники (вхід), B2B (вихід), API                   |
| Користувачі      | Гість, Покупець, Постачальник, Партнер, Адмін, Модератор |
| Зовнішні системи | Шипінг, Оплата                                           |

---

## Крок 2: Виділення Bounded Contexts

На основі аналізу виділено **6 обмежених контекстів**:

### 1. Product Catalog (Каталог товарів)

**Відповідальність:** Управління інформацією про товари

| Сутність | Опис                                 |
| -------- | ------------------------------------ |
| Product  | Товар з назвою, описом, зображеннями |
| Category | Категорія з ієрархією (parent/child) |
| Brand    | Бренд товару                         |

---

### 2. Supply Chain (Постачання)

**Відповідальність:** Управління постачальниками та запасами

| Сутність  | Опис                     |
| --------- | ------------------------ |
| Supplier  | Постачальник товарів     |
| Stock     | Залишки товару на складі |
| Warehouse | Склад зберігання         |

---

### 3. Pricing (Ціноутворення)

**Відповідальність:** Розрахунок цін, знижок, податків

| Сутність | Опис                                |
| -------- | ----------------------------------- |
| Price    | Ціна товару (з валютою, регіоном)   |
| Discount | Знижка (за категорією, промо)       |
| Tax      | Податок (за країною, типом покупця) |

---

### 4. Order Management (Замовлення)

**Відповідальність:** Процес покупки від кошика до доставки

| Сутність | Опис                    |
| -------- | ----------------------- |
| Cart     | Кошик користувача       |
| Order    | Оформлене замовлення    |
| Payment  | Інформація про оплату   |
| Shipment | Інформація про доставку |

---

### 5. User Management (Користувачі)

**Відповідальність:** Автентифікація, авторизація, ролі

| Сутність   | Опис                             |
| ---------- | -------------------------------- |
| User       | Користувач системи               |
| Role       | Роль (Гість, Покупець, Адмін...) |
| Permission | Дозволи для ролі                 |

---

### 6. Content Management (Контент)

**Відповідальність:** Блог, новини, відгуки

| Сутність | Опис            |
| -------- | --------------- |
| BlogPost | Стаття в блозі  |
| News     | Новина          |
| Review   | Відгук на товар |
| Rating   | Оцінка товару   |

---

## Крок 3: Зовнішні системи

Система інтегрується з:

| Система           | Тип      | Призначення             |
| ----------------- | -------- | ----------------------- |
| Payment Gateway   | Зовнішня | Обробка онлайн-платежів |
| Shipping Provider | Зовнішня | Доставка замовлень      |
| Warehouse System  | Зовнішня | Синхронізація запасів   |
| Accounting System | Зовнішня | Бухгалтерський облік    |

## Схема Bounded Contexts

```
┌─────────────────────────────────────────────────────────┐
│                    E-COMMERCE PORTAL                    │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  📦 PRODUCT CATALOG        🚚 SUPPLY CHAIN               │
│  ├── Product               ├── Supplier                 │
│  ├── Category              ├── Stock                    │
│  └── Brand                 └── Warehouse                │
│                                                         │
│  💰 PRICING                🛒 ORDER MANAGEMENT           │
│  ├── Price                 ├── Cart                     │
│  ├── Discount              ├── Order                    │
│  └── Tax                   ├── Payment                  │
│                            └── Shipment                 │
│                                                         │
│  👥 USER MANAGEMENT        📝 CONTENT MANAGEMENT         │
│  ├── User                  ├── BlogPost                 │
│  ├── Role                  ├── News                     │
│  └── Permission            ├── Review                   │
│                            └── Rating                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```
